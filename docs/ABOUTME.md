# About Me

## Who I Am
I'm a software developer based in Johannesburg, South Africa, currently studying software engineering at Belgium Campus iTversity, with freelance development work on the side. I'm a minimalist and a perfectionist — I'd rather do fewer things properly than many things half-finished. I'm self-motivated, curious, and regularly exploring new ideas and technologies outside of what I'm formally taught.

Software development and programming is both my current focus and what I want to do full-time going forward.

## Background
Before moving into software development, I spent 5 years as a freelance helpdesk technician. In that role I:
- Set up and configured WiFi networks for clients
- Built, deployed, and maintained servers
- Set up and administered virtual machines
- Diagnosed, fixed, and upgraded hardware and server infrastructure

That hands-on infrastructure background still shapes how I work today.
